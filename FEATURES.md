## 社区基础信息

### 1. list_communities - 列出所有支持的社区

**功能**：获取所有可查询的开源社区列表

**参数**：无

**示例问题**：
- 有哪些社区可以查询？
- 支持哪些开源社区？
- 列出所有社区

**返回示例**：
```
支持查询的社区列表：
  - openEuler
  - openGauss
  - MindSpore
  - CANN
  - openUBMC
  ...
```

---

### 2. get_community_list - 获取社区详细列表

**功能**：获取所有开源社区的完整列表（用于 PPT 展示）

**示例问题**：
- 获取社区全景图
- 所有社区的完整列表

**返回示例**：
```
共 25 个社区：
  - openubmc
  - openeuler
  - opengauss
  ...
```

---

### 3. get_metric_dict - 获取指标字典

**功能**：获取所有可用指标的名称、中文名、单位和定义

**参数**：无

**示例问题**：
- 有哪些衡量指标？

```
指标字典：

  [contributor_count] 贡献者数量 — 单位：count，说明：贡献者总数
  ...
```

## 社区健康度

### 4. get_community_health - 查询社区健康度

**功能**：查询指定社区的健康度综合评分及各子指标

**参数**：
- `community` (必填): 社区名称，如 openEuler、MindSpore、CANN 等

**示例问题**：
- openEuler 社区的健康度评分是多少？
- 查询 MindSpore 社区健康度
- CANN 社区的各项指标表现如何？

**返回示例**：
```
社区：openEuler
数据日期：2025-03-01
综合健康度评分：4.2 / 5.0

各指标评分（1-5分）及原始值：
  - 社区影响力(NSS)：4.5 分（原始值：850）
  - 下载量：4.0 分（原始值：125000）
  - Issue关闭率：4.3 分（原始值：0.85）
  ...
```

## PR 统计

### 5. get_prs_aggregate - PR 聚合统计

**功能**：获取 PR 总数、开启数、关闭数、合并数、响应时长、合并率等

**参数**：
- `community` (可选): 社区名称
- `start_time` (可选): 开始时间，格式 YYYY-MM-DD
- `end_time` (可选): 结束时间，格式 YYYY-MM-DD
- `interval` (可选): 时间粒度，支持 day/week/month

**示例问题**：
- openEuler 社区 2025 年的 PR 统计
- 查询 CANN 社区 2025-01-01 到 2026-01-01 的 PR 总数
- openEuler 2025 年每月的 PR 合并趋势
- MindSpore 社区的 PR 合并率是多少？

**返回示例（聚合）**：
```
PR 聚合统计：
  总数：525
  开启中：63
  已关闭：462
  已合并：420
  关闭率：0.88
  合并率：0.8
  平均首次响应时长：1.6 天
  平均关闭时长：5.8 天
```

**返回示例（按月）**：
```
PR 按month统计（共 8 个时间段）：
  2025-05: 总数 65, 合并 52, 合并率 0.8
  2025-06: 总数 72, 合并 58, 合并率 0.81
  2025-07: 总数 68, 合并 54, 合并率 0.79
  ...
```

### 15. get_cla_user_list - CLA 签署用户列表

**功能**：获取 CLA 签署用户分页列表

**参数**：
- `community` (可选): 社区名称，默认 openubmc
- `page` (可选): 页码，默认 1
- `page_size` (可选): 每页数量，默认 10
- `sign_type` (可选): 签署类型（企业/个人）
- `name` (可选): 按姓名模糊搜索
- `start_time` (可选): 开始时间
- `end_time` (可选): 结束时间
- `order_by` (可选): 排序字段，默认 created_at
- `order_dir` (可选): 排序方向，默认 DESC

**示例问题**：
- openubmc 社区已签署 CLA 的用户列表
- 查询企业 CLA 签署者
- 谁签署了 CLA？

**返回示例**：
```

  ...
```

## 技术说明

- 所有时间参数格式：`YYYY-MM-DD`（如 2025-01-01）
- 社区名称不区分大小写
- 时间粒度 `interval` 支持：`day`/`daily`、`week`、`month`
- 分页参数 `page` 从 1 开始
- 所有工具均为异步调用，自动处理并发
